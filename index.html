PPT 연동 도움말

PPT용 텍스트 복사는 문제·보기·정답·피드백을 슬라이드에 붙여넣기 좋게 정리합니다.

문제 이미지 저장은 대상·퀴즈형식·문제 지문만 16:9 이미지로 만듭니다. 날짜와 객관식 보기는 넣지 않습니다.

문제전체 이미지 저장은 문제 지문과 보기/힌트까지 포함해 1280×720 PNG로 저장합니다.

이미지 글자가 작거나 크면 글자 보통/크게/아주 크게를 선택하세요.

정답 이미지 저장은 정답과 해설만 따로 저장합니다.

빠른 사용법

PC에서는 왼쪽 목록, 가운데 편집, 오른쪽 피드백 모아보기가 동시에 보입니다.

모바일에서는 하단 버튼으로 목록·기록·피드백·원고 화면을 전환합니다.

PC와 모바일이 같은 자료를 보려면 상단의 room 코드를 동일하게 입력하세요. 예: church-sermon

Firestore 규칙 예시
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /rooms/{roomId}/sermons/{sermonId} {
      allow read, write: if request.auth != null;
    }
  }
}
로컬 백업 불러오기
